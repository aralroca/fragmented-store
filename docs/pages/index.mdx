- **Tiny**: Adds less than 1Kb to your bundle.
- **Easy**: Simple API and painless to setup.
- **Boilerplate Free**: No need for actions, reducers, middleware, selectors, providers, etc.
- **Performant**: Only renders when the expected property changes.
- **Tooling**: First class TypeScript support and [devtools](https://chrome.google.com/webstore/detail/teaful-devtools/lficdnnjoackdnaddfcgllmjdocofadc).
